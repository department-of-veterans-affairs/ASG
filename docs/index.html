About this page

A leading influencer in the VA Open API Pledge, ES-ASG creates developer-centric guidance for building critical API Management resources. Central to this guidance is the API Playbook. We encourage open collaboration and participation as we publish our evolving Wiki pages and welcome your input.

How to contribute:

  • Use the email link provided with each Wiki page. Please specify clearly the context of your comment.
  • Use the .mediawiki link provided with each Wiki page. Edit the .mediawiki file, inserting your comments. Upon saving your changes, create a Pull Request for review.
  • Use the Wiki Page Source folder to update or add any files in the folder. Upon saving your proposed changes, create a Pull Request for review.
